# MO JSP fragments (`partials`)

Reusable JSPF includes for `mo-home.jsp`.

| File | Purpose |
| --- | --- |
| `mo-layout-sidebar.jspf` | Left navigation and branding for MO routes. |
| `mo-layout-topbar.jspf` | Top bar actions (e.g. user menu hooks). |
| `mo-welcome-card.jspf` | Welcome / intro block for the main column. |
| `mo-modals.jspf` | Modal overlay markup (settings and shared TA modal structure). |

These fragments keep the main page declarative; styling is mostly shared TA CSS with MO overrides in `assets/mo/css/mo-home.css`.
