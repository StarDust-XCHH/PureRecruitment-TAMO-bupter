# MO styles

- **`mo-home.css`** — Module Organizer–specific rules on top of shared TA styles (`ta-layout`, `ta-components`, `ta-dashboard-cards`, `ta-jobs`, `ta-modal`) loaded by `pages/mo/mo-home.jsp`. Covers MO home layout tweaks, route panels, job board and applicant panels, and any MO-only components not covered by TA CSS.

Keep MO-specific visual changes here instead of editing TA stylesheets, so TA and MO shells stay separable.
